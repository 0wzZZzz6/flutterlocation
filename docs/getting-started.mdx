# Getting Started

In order to install the plugin, just add the latest version from [pub.dev](https://pub.dev/packages/location) in your `pubspec.yaml`.

```yaml
dependencies:
  location: ^5.0.0
```

You can then follow the different guide depending on which platform you wish to support.

- [Android](/installation/android)
- [iOS](/installation/ios)
- [macOS](/installation/macos)
- [Web](/installation/web)
